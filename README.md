Project Minotaur 设计思路

人物和怪物做一个实时的图层排序 ok

人物/怪物攻击时调用攻击区，受击区trigger，地形区非trigger，人物和怪物没有地形碰撞 ok

**碰撞体分为三个区**：攻击区（实体） 受击区（触发） 地形区（实体）

受击区现在是对象变量(object)，
角色以及攻击碰撞是场景变量(scene)
